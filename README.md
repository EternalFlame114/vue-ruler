# vue-ruler
> 基于vue 2.x的 刻度尺，只支持移动端
## 实例代码：
```
    体重
    <DLRuler :value="50" :min="30" :max="300" :onChange="changeWeight"></DLRuler>
    身高
    <DLRuler :value="60" :min="10" :max="100"></DLRuler>
    测试
    <DLRuler :value="35" :min="10" :max="120"></DLRuler>
```
